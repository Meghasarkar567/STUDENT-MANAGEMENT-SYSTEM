# 🎓 Student Management System

A menu-driven Student Management System built using Python.

## 📌 Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Check Result (Pass/Fail)
- Grade Calculation
- Student Ranking
- Show Class Topper
- Class Statistics
- Export Student Report to CSV
- Automatic Data Storage using JSON
